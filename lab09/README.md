# Лабораторна робота № 9

Тема: "Реалізація програмних модулів розгалужених та ітераційних обчислювальних процесів"

Мета: "Набути ґрунтовних вмінь і практичних навичок реалізації технології
       модульного програмування, застосування операторів С/С++ арифметичних, логічних, побітових операцій,
       умови, циклів та вибору під час розроблення статичних бібліотек, заголовкових файлів та програмних 
       засобів у кросплатформовому середовищі Code::Blocks"

Завдання: "Реалізувати функції розв’язування задач 9.1–9.3 як складових статичної бібліотеки libModulesПрізвище.а
           (проект ModulesПрізвище, створений під час виконання лабораторної роботи №8)"
